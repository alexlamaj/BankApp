# BankApp
Personal project for educational and training purpose. Basically, a Banking App which starts with a landing page, and then proceeds to the actual app.

Idea generated by 'Bankist App' by Jonas Schmedtmann's JavaScript Course

Development Server
-----------------------
Run ng start in Terminal for a dev server. Navigate to http://localhost:3000/. The application will automatically reload if you change any of the source files.

Test App
-----------------------
In order to test the app's features you can use these usernames (aj, jp, sj, cs) and pins (1111, 2222, 3333, 4444).

Build
-----------------------
Built entirely using HTML, CSS, and JavaScript, without the use of any tools or frameworks

Features
-----------------------
You can check and learn about the BankApp's features by reading the starting page --> features section
